Collapsing Alignment



This tool helps user to simplify output from alignment software such as MAFFT or Clustal Omega. User uploads file that contains aligned sequences, script takes first sequence as reference and collect data about all gaps ('-') in it. Then in other sequences software deletes amino acids, nucleotides or gaps that were aligned to these gaps in reference sequence. This software is meant to be step between any alignment software and alignment visualizer such as WebLogo 3
